Address Search Guide

When searching for the Address, keep the following tips in mind:

  • Try searching for a portion of the address:
    Example: "1401 Arch" instead of "1401 Arch Street"
  • Include any street directions in your search:
    Example: N, W, E, S
  • Verify the Address matches the Office of Property Assessment records: Visit OPA's website
  • Use the eClipse Building ID Map to find or confirm the correct Address, BIN, or Building Name before making a selection.

Search Options

  • OPA Account Number: Must be nine digits, without dashes or spaces (e.g. 001222234).
  • Building Identification Number (BIN): Found by clicking the building outline on the Building ID Map.
  • Suites or Unit Numbers: Identify a unit descriptor such as a condominium number. An address must be included in the Address field above. Suite values are case sensitive.
  • Building Name: Coming Soon! The Department is working to expand this search option. Some buildings may have names.
