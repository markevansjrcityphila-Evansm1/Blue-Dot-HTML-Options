Address Search Guide

Tips for Searching

  • Try searching for a portion of the address:
    Example: "1401 Arch" instead of "1401 Arch Street"
  • Include any street directions:
    Example: N, W, E, S
  • Verify Address with OPA records: Visit OPA's website
  • Check BIN or Building Name on the eClipse Building ID Map

Search Options

  • OPA Account Number: Must be nine digits, no dashes or spaces (e.g. 001222234).
  • Building Identification Number (BIN): Found by clicking the building outline on the Building ID Map.
  • Suites or Unit Numbers: Identify a unit descriptor. Must include the Address field. Suite values are case-sensitive.
  • Building Name: Coming Soon! Some buildings may have names.
